# devFinder

A job board for web and software developers.

## Author

Sydnee Haley ・[sydneehaley.com](http://sydneehaley.com)

## Overview

This is a full stack static application that uses Node.js, Express.js, and ejs template engine. Users can
create an account to find and save jobs they can apply to. Jobs are collected by web scraping/crawling.
